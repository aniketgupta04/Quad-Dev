# QuadShield

Parametric weekly income protection prototype for e-commerce delivery workers.

## Description

QuadShield is a web-based prototype designed to safeguard the livelihoods of delivery partners against unpredictable disruptions. By utilizing a parametric insurance model, the system eliminates the traditional claims process; instead, it triggers automatic payouts based on predefined events like heavy rain, traffic congestion, or platform outages. The application calculates personalized weekly premiums based on risk factors and simulates a seamless subscription and payout workflow.

## Getting Started

### Dependencies

* Any modern web browser (Chrome, Edge, Firefox, Safari).
* No external libraries or frameworks required.

### Installing

* Download or clone the project repository to your local machine.
* Navigate to the project folder.
* No additional installation or environment configuration is necessary.

### Executing program

* Locate the `index.html` file in the root directory.
* Open `index.html` in your preferred web browser.
* Follow these steps within the UI:
    1. Enter user details including location, income, shift, and speed.
    2. Click **Calculate Premium** to see your personalized rate.
    3. Click **Buy Subscription** to complete the payment simulation.
    4. Use the **Disruption Buttons** to simulate events and trigger automatic payouts.

## Help

* Ensure all input fields are populated before attempting to calculate the premium.
* Valid weekly income for this prototype must be between ₹3000 and ₹8000.
* You must complete the "Buy Subscription" step before the disruption triggers will function.
